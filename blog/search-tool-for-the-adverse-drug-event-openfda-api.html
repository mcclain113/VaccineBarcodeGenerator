Try Our New API Search Tool

February 23, 2025 by Barry McClain

Looking to search the FDA's reported adverse drug events (ADEs)? Pharmacyinformatics.net’s API URL constructor is a convenient way to query the openFDA ADE endpoint and present impressive visualizations of the data.

The search tool sets up the query by using the adverse drug event API as referenced on the open FDA site https://open.fda.gov/apis/. All disclaimers and limitations from the open FDA site apply to this data retrieval.

The data is then returned to the site and is displayed into customizable Plotly graphs.

Not to take anything away from the current interactive graphs provided by the FDA. They have a graphical tool too. Both tools have different options and functionality to consider.

What distinguishes this tool from others?

  1. Easy lookup fields to show how terms are stored.
  2. Make graphs easily with the tool by using “count by” versus “include in search” prompts.
  3. Show what is in a detailed safety report by using “count by” Unique Safety Report ID.
  4. For advanced users, the constructed URL is modifiable for additional syntax or adjustments.
