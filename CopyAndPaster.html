Automated Copy, Paste, and Tab from Excel to Another Application

This is a tool that simplifies having to copy and paste a lot of data from excel to any application. (Note: AutoHotkey works only on Windows)

There are two parts:
1) Create your list of contents in Microsoft Excel. There can be any number of columns and rows. I provide a sample Microsoft Excel file that has the file name pre-loaded in the AutoHotkey script (just add your data to the workbook). Just one last thing to modify.

2) A shortcut using AutoHotKey that then automatically starts the process until it hits a blank cell

Getting started:

  1. If you do not have it installed already, download and install the current version of AutoHotkey: https://autohotkey.com/download/
  2. SAVE the CopyandPaster file linked here anywhere (network drive, desktop, your documents, etc). For this demonstration, I recommend placing the file on your desktop: CopyandPaster AutoHotKey File
  3. SAVE a Microsoft Excel file that contains what you want copied and pasted to a destination application. For this demonstration, I have provided a sample file (named copyandpaster.xlsx). I recommend placing the file on your desktop: CopyandPaster Excel File
  4. To EDIT the script, right-click the CopyandPaster file and select Edit Script
  5. Suggested modifiable items are highlighted
    1. Update the path name of your MS Excel workbook. IF you are using the instructions above and using the same desktop path and file name, then ignore.
    2. Update number of columns on the spreadsheet. 0, 0 = 1 column. 0, 1 = 2 columns. 0, 2 = 3 columns. And so on. By default, 2 columns are set. Computers like to start counting at 0.
    3. Update the speed of the script. By default, it is set to 500 microseconds (0.5 seconds). Depending on what is receiving the data, it might not be able to handle a fast velocity.

Using the script:

  1. Simply double clickthe CopyandPaster file. Once it is running, the script will open an instance of the Excel workbook. There is nothing to do with the workbook.
  2. Once your cursor is in the destination application, press this keystroke combo to activate the script: Windows button + b
  3. When the script completes, it will display a window stated it is finished
  4. If the script falls off track and needs to be aborted immediately, just press the escape key (Esc) or right-click the running script in your tray and exit
